POSITION SUMMARY:
We are seeking an experienced Lead DevOps Engineer to architect and manage our cloud infrastructure. The ideal candidate will have deep expertise in automation, CI/CD pipelines, and infrastructure as code. You will lead a team of engineers and drive our cloud migration and DevOps transformation initiatives.

ESSENTIAL DUTIES AND RESPONSIBILITIES:
- Design and implement scalable cloud infrastructure on AWS and Azure
- Build and maintain CI/CD pipelines using Jenkins, GitLab CI, and GitHub Actions
- Automate infrastructure provisioning using Terraform and Ansible
- Implement containerization strategies with Docker and Kubernetes
- Monitor system performance and reliability using observability tools
- Lead incident response and post-mortem analysis
- Establish DevOps best practices and standards across engineering teams
- Mentor and guide junior DevOps engineers
- Collaborate with security team on compliance and vulnerability management
- Optimize cloud costs and resource utilization
- Participate in on-call rotation for production support

REQUIRED QUALIFICATIONS:
- Bachelor's degree in Computer Science, Information Technology, or related field
- 6+ years of experience in DevOps, SRE, or cloud infrastructure roles
- Strong expertise in AWS services (EC2, S3, Lambda, ECS, RDS, CloudFormation)
- Proficiency with infrastructure as code tools (Terraform, CloudFormation, Pulumi)
- Hands-on experience with container orchestration (Kubernetes, ECS, Docker Swarm)
- Expert knowledge of CI/CD tools (Jenkins, GitLab CI, CircleCI, GitHub Actions)
- Strong scripting skills in Python, Bash, or PowerShell
- Experience with configuration management tools (Ansible, Chef, Puppet)
- Deep understanding of networking, security, and cloud architecture
- Experience with monitoring and logging solutions (Prometheus, Grafana, ELK)
- Strong troubleshooting and problem-solving abilities
